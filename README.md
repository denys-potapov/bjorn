# glagoly - online voting platform

## Requirments

Install npm or update npm to latest version:

    sudo npm install npm -g

After you get them insall npm dependencies:
    
    npm install

Insall bower globally:

    npm install -g bower
     
Install bower dependencies:

    bower install

Run local

    npm run start
